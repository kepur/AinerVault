# 06. ainer-model-router

## 定位
模型路由与资源调度中心（DispatchDecision 权威层）。

## 核心职责
- 按任务类型、预算、SLA、节点健康选择后端。
- 维护熔断/降级/回退链。
- 输出可审计的 `dispatch_decision`。

## 输入
- 任务画像、质量目标、预算约束、实时资源状态。

## 输出
- `route.decision.made`、`route.degraded`。

## 上下游
- 上游：`05.ainer-planner.md`、`02.ainer-orchestrator.md`。
- 下游：`07.ainer-worker-hub.md`。

## 边界
- 可写：路由审计、策略命中记录。
- 不可写：业务语义与剧情决策。

## 失败与回滚
- 无可用后端：返回降级建议并标记 `route.degraded`。
- 后端不稳定：触发熔断并切 fallback 链。
