# 09. ainer-worker-audio

## 定位
音频生成执行器（TTS/BGM/SFX/Ambience）。

## 核心职责
- 执行音频生成与标准化。
- 产出准确时长/时间戳元数据（时序权威来源之一）。
- 输出音频产物供 06/12 使用。

## 输入
- 音频任务、角色语音配置、素材引用。

## 输出
- `job.succeeded/job.failed`（主状态）、`worker.audio.completed`（执行明细）、`audio_timing_metadata`、`artifact_meta`。

## 上下游
- 上游：`07.ainer-worker-hub.md`。
- 下游：`06 skill` 时间线合成、`12.ainer-composer.md`。

## 边界
- 不做全片合成。
- 不直接推进 run 阶段终态。

## 失败策略
- 音轨缺失：返回可降级标签（例如 mute/placeholder）。
- 时长异常：标记 `REVIEW_REQUIRED` 并阻断自动发布。
